# Quick Start — Sprint 2 Quality

## Local checks
```bash
pnpm lint
pnpm typecheck
pnpm build
pnpm start
````

## Playwright (once implemented)

```bash
pnpm test:e2e
```

## Release smoke checklist (manual)

* Login
* Create a task
* Start/stop a session
* Dashboard totals change
* Theme persists
* PWA installable
* Offline page works
